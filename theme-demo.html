MCP Inspector - Theme Customization Demo

Quick Start Example
The simplest way to customize the font family across the entire component library:
import MCPInspector from 'mcp-inspector'; function App() { const customTheme = { typography: { fontFamily: '"Inter", "Roboto", "Arial", sans-serif', }, }; return ( <MCPInspector url="wss://your-mcp-server" themeCustomization={customTheme} /> ); }
Comprehensive Customization
Example showing colors, typography, spacing, and component-specific customizations:
const customTheme = { typography: { fontFamily: '"Poppins", "Inter", "Roboto", sans-serif', h1: { fontWeight: 700, letterSpacing: '-0.02em' }, body1: { fontSize: '14px', lineHeight: 1.6 }, }, palette: { primary: { main: '#6366f1', // Indigo light: '#818cf8', dark: '#4f46e5', }, secondary: { main: '#8b5cf6', // Purple }, }, shape: { borderRadius: 8, }, custom: { colors: { accent: '#f59e0b', // Amber muted: '#6b7280', }, components: { button: { borderRadius: 6, fontWeight: 600, }, card: { borderRadius: 12, shadow: '0 4px 6px -1px rgba(0, 0, 0, 0.1)', }, }, }, };
Using Predefined Presets
The library comes with several predefined theme presets for common use cases:
import { themePresets } from 'mcp-inspector'; // Modern theme with Inter font and contemporary styling <MCPInspector themeCustomization={themePresets.modern} /> // Compact theme with condensed fonts and tighter spacing <MCPInspector themeCustomization={themePresets.compact} /> // Rounded theme with highly rounded corners <MCPInspector themeCustomization={themePresets.rounded} />
Dark Mode with Custom Colors
Enable dark mode with customized colors optimized for dark themes:
const darkCustomTheme = { palette: { primary: { main: '#60a5fa', // Lighter blue for dark mode }, background: { default: '#0f172a', paper: '#1e293b', }, text: { primary: '#f8fafc', secondary: '#cbd5e1', }, }, }; <MCPInspector darkMode={true} themeCustomization={darkCustomTheme} />
Advanced Material-UI Overrides
For fine-grained control, use Material-UI's override system:
const advancedTheme = { overrides: { MuiButton: { root: { textTransform: 'none', borderRadius: '6px', fontWeight: 500, '&:hover': { boxShadow: '0 4px 8px rgba(0, 0, 0, 0.12)', }, }, }, MuiTextField: { root: { '& .MuiOutlinedInput-root': { borderRadius: '8px', '&.Mui-focused fieldset': { borderColor: '#6366f1', borderWidth: '2px', }, }, }, }, }, };
Key Features
  • Style Isolation: Styles are scoped and won't affect your parent application
  • TypeScript Support: Full type safety for all customization options
  • Predefined Presets: Ready-to-use themes for common design patterns
  • Flexible API: From simple font changes to complex Material-UI overrides
  • Dark Mode Support: Built-in dark mode with custom color support
  • Component-Specific Styling: Fine-tune individual component styles
